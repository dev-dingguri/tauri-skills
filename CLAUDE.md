## Project Overview

A collection of AI coding agent skills for Tauri v2 desktop app development on Windows.

## Language Policy

All content in this repository must be written in English:
- Skill files (SKILL.md), documentation, code comments, examples
- Commit messages (title and body)
- Git hooks and Claude Code hooks enforce this automatically

## Confidentiality

Skills must be generic and reusable. Do not include identifiable project names, product names, or organization names in any file. Use generic placeholders (e.g., "MyApp", "your-app") in examples instead.

## Skill Conventions

- Each skill lives in `skills/<skill-name>/SKILL.md`
- Examples in skills should use generic, non-identifiable names
